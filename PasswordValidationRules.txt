
Password Validation Rules

- Minimum Length: Password must be at least 8 characters long.
- Uppercase Letter: Password must contain at least one uppercase letter (e.g., A-Z).
- Lowercase Letter: Password must contain at least one lowercase letter (e.g., a-z).
- Numeric Digit: Password must contain at least 2 numbers (e.g., 0-9).
- Special Character: Password must include at least 1 special character (e.g., !@#$%^&*()_-+={[}]|:;"'<>,.?/`~).
- Unique Characters: Password must contain at least 5 unique characters (e.g., no repeating the same character excessively).
- No Common Words: Password should not contain easily guessable words or sequences (e.g., password, 123456, qwerty).
- Password History: Password history of 3 previous passwords cannot be defined in the reset.
- Allowed Failed Attempts: User will be blocked after 5 failed login attempts.
- Lockout Time: After blocking the user, the user will be in time-out for 15 minutes.
